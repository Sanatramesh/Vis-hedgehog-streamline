Compilation Instructions
The program consists of two files:
1. VectorFieldPlot.cpp ----> It contains code for plotting hedgehog and streamlines.

   To compile do:
   make VectorField

   and run using command:
  ./vectorfield <datafile with path> 

  To remove the binary file do:
  make clear

      ##################### Useful Keys #####################
      z ---> Zoom In
      x ---> Zoom Out
      c ---> Change between steamlines and hedgehog

2. VectorField.cpp  ----> It contains all the utility functions used in VectorFieldPlot.cpp
